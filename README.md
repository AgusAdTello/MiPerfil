## 📖 Descripción

Este es mi portafolio personal, diseñado para mostrar mis habilidades, experiencia y proyectos destacados. Está construido como una *Single Page Application* (SPA) moderna, con un diseño responsivo, modo oscuro por defecto y animaciones fluidas.

El objetivo de este sitio es servir como carta de presentación para oportunidades laborales, permitiendo a los reclutadores ver mi código, mis demos en vivo y contactarme fácilmente.

## ✨ Características Principales

* **Diseño Moderno:** Interfaz oscura ("Dark Mode") construida con **Tailwind CSS**.
* **Navegación Fluida:** Scroll suave entre secciones (Inicio, Proyectos, Habilidades, Contacto).
* **Showcase de Proyectos:** Sección destacada para mostrar aplicaciones como *AgusCine*.
* **Formulario de Contacto Funcional:** Integración con **Formspree** para recibir mensajes directamente al email sin necesidad de backend propio.
* **Iconografía:** Uso de `lucide-react` para iconos limpios y consistentes.
* **Totalmente Responsive:** Adaptable a móviles, tablets y escritorio.

## 🛠️ Tecnologías Utilizadas

* **Frontend:** React 19
* **Build Tool:** Vite
* **Estilos:** Tailwind CSS v4
* **Iconos:** Lucide React

## 🚀 Instalación y Ejecución Local

Sigue estos pasos para correr el proyecto en tu computadora:

1.  **Clonar el repositorio:**
    ```bash
    git clone [https://github.com/AgusAdTello/mi-portfolio.git](https://github.com/AgusAdTello/mi-portfolio.git)
    cd mi-portfolio
    ```

2.  **Instalar dependencias:**
    ```bash
    npm install
    ```

3.  **Iniciar el servidor de desarrollo:**
    ```bash
    npm run dev
    ```

## 📂 Estructura del Proyecto

El código principal se encuentra en `src/App.jsx`, manteniendo una estructura simple y fácil de mantener:

* **Hero Section:** Presentación personal y enlaces a redes.
* **Proyectos:** Tarjetas interactivas con links a repositorios y demos.
* **Habilidades:** Grid visual con el stack tecnológico (.NET, SQL Server, React, etc.).
* **Contacto:** Formulario validado y conectado a servicio de email.
